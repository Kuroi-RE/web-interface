## Run Locally

Clone the project

```bash
  git clone https://github.com/Kuroi-RE/web-interface.git
```

Go to the project directory

```bash
  cd web-interface
```

Install dependencies

```bash
yarn add or simply yarn
```

Start the server

```bash
yarn run dev
```
